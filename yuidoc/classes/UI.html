(
  • evt
  • ui
)

Defined in /Users/philippeperret/Sites/table_narration/js/ui/ui.js:119

Méthode appelée lorsque l'on glisse un outil sur la table (principalement pour créer une nouvelle fiche).

Notes

  • Les deux utilisations principales sont :
    • La création d'une nouvelle fiche (à l'aide d'un outil-fiche)
    • La sortie d'une fiche d'un de ses parents.

Parameters:

  • evt Event

    Drop event

  • ui Object

    Set jQuery transmis par droppable.
