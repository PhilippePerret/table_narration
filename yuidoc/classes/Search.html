async

Defined in /Users/philippeperret/Sites/table_narration/js/Search/>Search.js:88

= Main = Fonction principale pour invoquer une recherche

Notes

  • Cette méthode peut être utilisée par le raccourci find(...) défini en bas de ce module.
  • C'est cette méthode qui est appelée par le bouton “Rechercher” dans le formulaire de recherche.

Parameters:

  • text String | Regexp

    Le texte à chercher exprimé en raw text ou comme une expression régulière

  • params Object

    Les paramètres optionnels de la recherche

Example:

Search.text("Ce texte à chercher") Search.text(/(un|des) textes?/)
